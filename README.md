# Homebrew Tap for Krusty

This tap contains the Homebrew formula for [Krusty](https://github.com/BurgessTG/Krusty), a terminal-based AI coding assistant.

## Installation

```bash
brew tap BurgessTG/tap
brew install krusty
```

## Update

```bash
brew update
brew upgrade krusty
```

## Uninstall

```bash
brew uninstall krusty
brew untap BurgessTG/tap
```
